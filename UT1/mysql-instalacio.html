Instal·lació de MySQL

Els paquets de distribució per a MySQL estan disponibles per a diverses plataformes, incloses les distribucions de Linux, Windows i macOS. Aquests paquets proporcionen una manera flexible i ràpida de començar a utilitzar MySQL. L'última versió LTS de MySQL és la 8.4.

Instal·lació en Linux

En Linux tenim diverses possibilitats per a fer la instal·lació:

  • Descarregar el paquet amb la versió de MySQL per al nostre sistema operatiu de https://dev.mysql.com/downloads/.
  • Des de l'anterior pàgina es poden configurar els repositoris oficial de MySQL compatibles con distintes distribucions Linux.
  • Utilitzar els repositoris de la distribució per a fer la instal·lació. Moltes distribucions opten per incorporar el fork MariaDB Server front MySQL Community Edition.
  • Compilar la nostra pròpia versió des del codi font.

Instal·lació des del repositori oficial Yum

Primerament s'ha d'obtindre l'arxiu rpm que permet la configuració del repositori oficial. El nom de l'arxiu té el format:

mysql84-community-release-{platform}-{version-number}.noarch.rpm

{platform}: El codi de la plataforma, com ara el7, el8, el9, fc38, fc39, o fc40. 'el' representa Enterprise Linux, 'fc' per a Fedora, i acaba amb el número de versió.

{version-number}: Versió de la configuració del repositori de MySQL.

Instal·la l'RPM descarregat al SO:

$> sudo dnf localinstall mysql84-community-release-{platform}-{version-number}.noarch.rpm

Instal·la MySQL amb el següent comandament:

$> sudo dnf install mysql-community-server

Açò instal·la el paquet de MySQL server (mysql-community-server) i els paquets per al components requerits per al servidor, incloent els paquets del client (mysql-community-client), els missatges comuns d'errors i el conjunt de caràcters per al client i el servidor (mysql-community-common), i les libreries compartides (mysql-community-libs).

Per arrancar el servidor de MySQL:

$> systemctl start mysqld

Per a comprovar l'estat del servidor:

$> systemctl status mysqld

Al primer inici del servidor es realitzen les següents tasques:

  • Inicialització del servidor.
  • Generació d'un certificat SSL i els fitxers de clau al directori de dades.
  • S'habilita validate_password. La política de contrasenya predeterminada implementada per validate_password requereix que les contrasenyes continguin almenys una lletra majúscula, una lletra minúscula, un dígit i un caràcter especial, i que la longitud total de la contrasenya sigui d'almenys 8 caràcters.
  • Es crea un compte de superusuari 'root'@'localhost'. S'estableix una contrasenya per al superusuari i s'emmagatzema al fitxer de registre d'errors /var/log/mysqld.log.

Primers passos amb MySQL

Per comprovar el funcionament del servidor accedim a ell amb el client:

$> mysql -u root -p

Podem passar un script SQL perquè s'execute en el moment d'accedir amb el client:

$> mysql -u root -p < "arxiu.sql"

Algunes ordres bàsiques són:

SHOW databases; -- mostra les BD existents al servidor
USE nom_bbdd; -- accedeix a una BD
SHOW tables; -- mostra les taules d'una BD
DESCRIBE nom_taula; -- mostra els detalls d'una taula (columnes, tipus, claus...)
CREATEA DATABASE nom_bbdd; -- crea una BD
DROP DATABASE nom_bbdd; -- esborra una BD

Exercici

Criteris d'avaluació 1f, 1g, 1h, 1i, 1j.

Instal·lació de MySQL Community Edition

  • Aquesta pràctica es realitzarà dins el Learner Lab d'AWS del mòdul. Documenta amb captures de pantalla els passos seguits.
  • Crea una instància d'EC2 amb les mínimes especificacions necessàries per realitzar correctament la instal·lació de MySQL Community Edition. Detalla tant les especificacions de la instància com els requiriments de MySQL.
  • El SO de la instància EC2 serà una distribució Linux de la teua elecció. Justifica el motiu de la vostra elecció.
  • La configuració de la instància es realitzarà via SSH des de l'equip client de l'administrador. Genera un parell de claus SSH en AWS i assigna-les a la instància. Assegura't que l'accés SSH només és permés des de la IP de l'equip client.
  • Realitza la instal·lació seguint els passos necessaris. Posa en marxa el servidor i fes que el servei es pose en marxa automàticament després de l'inici de la instància.
  • Una vegada realitzada la instal·lació mostra el contingut de l'arxiu de log d'errors i connecta't amb el client de MySQL amb l'usuari 'root@localhost'. Si tot funciona correctament, ix del client.
  • Anem a crear una BD d'exemple i a inserir dades a ella. Per això, descarrega algunes de les BD d'exemple de la pàgina https://dev.mysql.com/doc/index-other.html i executa l'script. Executa alguna de les ordres bàsiques per mostrar que s'han creat correctament les dades.
  • Després de finalitzar la pràctica atura la instància EC2 i crea un snapshot del volum EBS associat a aquesta. Una vegada creat l'snapshot destrueix el volum EBS.
